# Auto CPU Spoofer

A lightweight utility to automatically replace the cosmetic CPU name string in Windows.

## Features
 - Changes CPU name string without restart required
 - Lightweight and fast command line prompt program
 - Supports patch on Windows startup

## Prerequisites
 - Administrator is required (Necessary to modify registry keys)

## Installation & Usage
1. Download the latest release from the [Releases](https://github.com/df-001/auto-cpu-spoof/releases) page.
2. Open auto-cpu-spoof.exe
3. Accept UAC admin prompt.
4. Verify change in Task manager and close program.
